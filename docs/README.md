# 宝宝胎动记录器 ❤️

这是一个简单的网页应用，专为我怀孕28周零1天的老婆设计，用于记录胎动。支持PC、手机、iPad自适应。

## 功能概述
- **记录 & 保存**：点击按钮累加胎动次数，保存时记录日期（MM/DD/YYYY）和时间（HH:MM），存到浏览器localStorage。
- **浏览历史**：查看过往记录，按日期降序显示。

## 安装 & 使用
1. 下载整个文件夹。
2. 在浏览器打开 `index.html`。
3. 点击“记录胎动 +1”按钮计数，点“保存记录”保存。
4. 点“查看历史”浏览过去数据。

## 技术栈
- HTML5
- CSS3 (媒体查询自适应)
- JavaScript (localStorage存储)

## 开发笔记
- 数据持久化：用localStorage，浏览器清缓存会丢失（未来可加云同步）。
- 扩展想法：加定时提醒、数据导出CSV、分享功能。
